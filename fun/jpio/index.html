JPIO

What is it

Json Python IO is a command line utility to read/write json data. It started out as a simple idea to read and write json files via command line. As I started working on it, I soon wonder how much I can pushed it. It has now become a query/transformation language for json.

Existing implementations

I found a few existing implementation that are good, like ‘json’ (written in node), and jq. However, I wanted a implementation I can modify and add new stuffs to. So as usual, I decided to tackle the problem.
