Lua tutorial

Welcome to the Lua tutorial. This is not a tutorial about Lua as a programming language; There are plenty of those around. Instead, I will assume you know the basic of lua and how Objects are modeled in Lua.

This tutorial will cover how to write C++ programs that interact with Lua. There are many reasons why you would want to do this but usually is to allow for extension and modification without recompilation.

This Lua tutorial is constructed using information I searched over the internet and may not be the best way to do certain things. Most of them are stuffs that I constructed using my limited knowledge of Lua. They should be usable for more scenario.

The things I will be covering

  • Setting up
  • Running a Lua function from C++
  • Running a C++ function from Lua
  • Object-oriented in Lua (Coming Soon ?)
  • Accessing C++ objects from Lua
  • Package a function in Lua

All the code for this tutorial can be found at github in the tutorial subfolder. In the case that there is a difference between the code in the repo and the code in the tutorial, always look at the code in the repo instead. They are always more updated and always compilable(at least on my machine).
